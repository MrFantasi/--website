# --website
 SURODIN adalah meme coin yang terinspirasi dari kisah nyata Simbah Surodin, seorang lelaki tua  yang hidup terpinggirkan dan tidak dipedulikan keluarganya. Proyek ini dibuat sebagai bentuk  penghormatan dan upaya menyelamatkan tanah warisan yang direbut oleh mereka yang tidak  pernah merawatnya.
